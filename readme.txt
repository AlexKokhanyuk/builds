Maven options

~start from the home progect directory:
mvn clean - to clean work directory
mvn install - to run build (jar in ~\target)
mvm test - to run test 


Gradle options

~start from the home progect directory:
gradle build - to run build
gradle jar - to make jar (jar in ~\build\libs)
gradle test - to run test
gradle clean - to clean work directory


Ant options
~start from the home progect directory:
ant  junit - to run test
ant compile - to make class
ant dist - to make jar (jar in ~\dist\lib)
ant clean - to clean work directory
